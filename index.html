This is a demo page for my HTML board for chess and (international) checkers. It can show a position (using FEN), replay a chess game (from PGN) or let you select valid moves. It can also forward your move to an external javascript helper for online games, you just have to write the server and helpers ;)

Credits

  • It started as a fork of jchess but I ended up rewriting everything but the parts of PGN parser.
  • Piece images from Wikimedia.
  • It was inspired by the look of Chess By Post. The PGN parser was fixed to handle CBP's PGN export.
